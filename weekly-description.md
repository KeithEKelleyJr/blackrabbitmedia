# Weekly Graphic Recreation Guide (`Weekly.png`)

Use this as the repeatable template to rebuild the weekly square post.

## 1) Canvas + Style

- **Size:** 1080 x 1080 px (square)
- **Format:** PNG
- **Theme:** black background, white text, orange accents
- **Estimated accent color:** `#F07000` (close match from source)
- **Main structure:** centered stacked header + 5 horizontal event rows

## 2) Top Header Block

Order from top to bottom:

1. **BLACK RABBIT** (largest, bold, white)
2. **SPEAKEASY** (smaller, white)
3. **UPCOMING EVENTS** (all caps, white)
4. **HAPPY HOUR EVERY DAY 4 TO 6** (white)
5. **BLACKRABBITSPEAKEASY.com** (white)

Keep this block centered and leave visual gap before the event rows.

## 3) Weekly Rows Layout (Reusable)

Create **5 rows**, each with:

- **Left column:** month/day (ex: `May` + `5`) in orange
- **Right column:** event title + 1-2 description lines in white

Consistent spacing:

- Equal vertical gaps between rows
- Same left edge for all row text blocks
- Same typography scale pattern in each row

## 6) Weekly Update Checklist

1. Duplicate last week’s design file.
2. Update month/day values in left column.
3. Update event title + details in right column.
4. Keep spacing/alignment identical to preserve branding.
5. Export PNG at 1080 x 1080 as `Weekly.png`.
